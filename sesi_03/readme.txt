Panduan penggunaan aplikasi:
Di sesi 3, saya mempelajari tentang looping, antara lain:
    - For Loop
    - While Loop
    - Do-While Loop
    - Nested Loop

1. For Loop, dimana akan melakukan loop sebanyak x kali, dengan x adalah sebuah nilai yang
   bisa kita tentukan berapa kali. Selain itu juga terdapat for each yang melakukan looping
   pada array atau list sebanyak jumlah elemen di dalamnya.

2. While Loop, dimana akan melakukan looping selama kondisi terpenuhi. Kita harus berhati-hati
   ketika menggunakan while loop untuk mengecek kembali apakah perlu ada kondisi false serta 
   bagaimana agar mencapai kondisi false sehingga tidak menimbulkan infinite loop

3. Do-While Loop, sebenarnya mirip dengan While Loop, perbedaanya dalam do while kondisi pertama
   pasti akan dijalankan dan setelah itu baru akan memeriksa kondisi while.
   Contoh i = 1, while i <= 0.
   Jika di While Loop, sama sekali tidak akan menjalankan perintah di dalam while, karena
   1 tidak lebih kecil sama dengan 0. Namun pada Do-While Loop akan satu kali menjalankan perintah di dalam
   do karena saat pertama belum memeriksa kondisi While.

4. Nested Loop, yaitu membuat loop baru di dalam loop. Misalkan For Loop di dalam For Loop.
