Lalu untuk menjalankan program tersebut adalah dengan menjalankan
terminal/command prompt di folder sesi_01, lalu jalankan perintah
"javac <NamaFile>.java" (tanpa tanda kutip), maka akan muncul file <NamaFile>.class, lalu jalankan
"java <NamaFile>" (tanpa tanda kutip). Contohnya ketika menjalankan Hello.java adalah seperti ini:

javac Hello.java
java Hello
