Saya belajar mengenai relasi antar table. Terdapat 3 jenis relasi, yaitu:
1. One to one, misal setiap warga negara hanya bisa memiliki 1 KTP
2. One to many, misal satu orang bisa memiliki beberapa motor, tetapi
   motor tersebut berdasarkan BPKB hanya milik satu orang.
3. Many to many, misal setiap mahasiswa bisa mengambil beberapa mata kuliah
   dan setiap mata kuliah juga menampung beberapa mahasiswa.

Saya juga belajar Data Definition Language (DDL) dan Data Manipulation Language (DML).

- CREATE DATABASE
- CREATE TABLE
- ALTER TABLE
- DROP TABLE
- CREATE INDEX
- DROP INDEX

Sedangkan DML meliputi:
- SELECT
- INSERT
- UPDATE
- DELETE
