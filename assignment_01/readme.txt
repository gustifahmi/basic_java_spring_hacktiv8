1. LuasLingkaran.java (Latihan 1)
   Pertama-tama saya sudah menginisasi nilai pi, yaitu 3.14, lalu saya juga sudah
   menginisiasi variabel kosong, yaitu r untuk menyimpan nilai jari-jari lingkaran dan
   luas untuk menyimpan nilai luas lingkaran. Lalu program akan meminta input nilai
   jari-jari dari user. Setelah user memasukkan nilai jari-jari maka akan mengkalkulasikan
   luas lingkaran dengan mengalikan pi dengan Math.pow(r, 2) yang berarti r kuadrat. Lalu
   hasilnya akan diprint dengan 2 angka di belakang koma.

2. Aritmatika.java (Latihan 2)
   Terdapat 3 variabel yang sudah diinisiasi beserta valuenya, yaitu a bernilai 12,
   b bernilai 8, dan c bernilai 5. Selanjutnya terdapat 6 kali operator aritmatika, yaitu:
   1) a + b - c -> seharusnya mendapatkan hasil 15
   2) a * b / c -> seharusnya mendapatkan hasil 19
   3) a + b * c -> seharusnya mendapatkan hasil 52
   4) a + b / c -> seharusnya mendapatkan hasil 13
   5) (a + b) * c -> seharusnya mendapatkan hasil 100
   6) (a - b) * c -> seharusnya mendapatkan hasil 20
   Program sudah dicoba dan keenamnya mendapatkan hasil yang sesuai.

3. IncrementDecrement.java (Latihan 3)
   Terdapat 4 variabel yang sudah diinisiasi beserta valuenya, yaitu a bernilai 0, b bernilai 0,
   c bernilai 9, dan d bernilai 9. Pada print ke-1 saya mengikuti perintah increment dan decrement
   pada soal. Sedangkan pada print ke-2, ke-3, dan ke-4, saya harus mendapatkan hasil print yang sesuai
   dengan soal dengan hanya menggunakan increment dan decrement. Setelah melakukan increment dan decrement
   sedemikian rupa saya mendapatkan hasil print yang sesuai di seluruh print yang sebenarnya pola
   increment dan decrement sama dengan print ke-1.

4. PerbandinganTrueFalse.java (Latihan 4)
   Terdapat 4 variabel yang sudah diinisasi beserta valuenya, yaitu a bernilai 10, b bernilai 8,
   c bernilai 12, dan d bernilai 5. Terdapat 10 kali print yang harus saya buat dan print 1-6 harus
   bernilai true, sedangkan print 7-10 harus bernilai false. Saya hanya bisa menggunakan 4 variabel
   tersebut, yaitu a, b, c, dan d. Sedangkan 10 print tersebut menggunakan operator perbandingan
   yang berbeda-beda dengan rincian sebagai berikut:
   1) > ---> Saya menggunakan a > b sehingga menghasilkan true
   2) < ---> Saya menggunakan d < a sehingga menghasilkan true
   3) >= ---> Saya menggunakan c >= b sehingga menghasilkan true
   4) <= ---> Saya menggunakan d <= c sehingga menghasilkan true
   5) == ---> Saya menggunakan a == 2*d sehingga menghasilkan true
   6) != ---> Saya menggunakan b != c sehingga menghasilkan true
   7) > ---> Saya menggunakan b > a sehingga menghasilkan false
   8) <= ---> Saya menggunakan c <= d sehingga menghasilkan false
   9) == ---> Saya menggunakan b == c sehingga menghasilkan false
   10) != ---> Saya menggunakan a != 2*d sehingga menghasilkan false
   Program sudah dicoba dan hasilnya sesuai dengan yang diharapkan.

5. PenjumlahanXY.java (Latihan 5)
   Saya menginisiasi y1 dan y2 yang nantinya akan digunakan untuk mengambil input dari
   user. Lalu setelah mengambil input y1 dan y2 dari user, maka akan melakukan kalkulasi
   sesuai dengan soal sehingga mendapatkan X1 dan X2, dengan perhitungan sebagai berikut:
   X1 = (y1 + y2) * (y1 + y2)
   X2 = (y1 % 4) * y2
   
   Selanjutnya program print hasil X1 dan X2, lalu program akan mengecek 3 kondisi
   seperti di soal, yaitu:
   1. X1 besar sama dengan X2
   2. X2 besar sama dengan X1
   3. X1 mod 4 == ++X2 mod 5
   Setelah itu program akan print hasil pengecekan 3 kondisi tersebut.
