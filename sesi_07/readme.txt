Di sesi 7 ini, saya mempelajari tentang algoritma. Algoritma adalah instruksi yang dibuat
untuk menyelesaikan suatu persoalan.

Algoritma yang dipelajari di sesi ini, antara lain:
1. Selection Sort
   Selection Sort adalah algoritma sort yang caranya adalah menukar index 0 dengan nilai
   terkecil, lalu dilanjutkan ke index 1 dan seterusnya sampai index terakhir.

2. Binary Search
   Binary search adalah melakukan proses pencarian dengan membagi array menjadi dua
   dengan menggunakan nilai tengah (middle), lalu mengecek apakah angka yang dicari
   sama dengan middle, jika sama maka kita tahu index yang diinginkan. Namun jika tidak,
   maka kita mencari lagi, jika lebih kecil dari middle, maka akan mencari di paruh awal,
   sedangkan jika lebih besar dari middle, maka akan mencari di paruh akhir. Begitu seterusnya
   sampai angkanya ketemu di middle.
