Panduan penggunaan aplikasi:

Di sesi 22 ini, saya mempelajari tentang Heroku. Heroku merupakan sebuah tools sebagai lokasi deploy
program kita yang bisa digunakan secara gratis. Langkah-langkahnya adalah sebagai berikut:
   1. Di local, buat sebuah folder baru khusus untuk demo heroku(di luar dari folder seluruh sesi Java ini)
   2. Buat sebuah project spring boot sederhana, misalnya menampilkan tulisan 'Hello World" di
      halaman index. (Atau bisa juga menggunakan project yang pernah dibuat, yang terpenting
      tidak ada error)
   3. Buat repo Github baru khusus untuk demo ini.
   4. Hubungkan folder demo tersebut ke Github repo yang telah dibuat.
   5. Buat akun Heroku terlebih dahulu (Saya sebelumny sudah mempunyai akun Heroku, sehingga bisa
      langsung ke langkah berikutnya).
   6. Buat sebuah app baru.
   7. Buka halaman app tersebut, lalu masuk ke tab "Deploy"
   8. Pilih "Github" pada Deployment Method
   9. Terdapat input nama repo, masukkan nama repo dari demo heroku, klik Search
   10. Setelah hasil pencarian keluar, klik connect.
   11. Setelah berhasil connect, klik "Deploy Branch", pada bagian Manual Deploy
   12. Setleah berhasil deploy, kita bisa melihat hasilnya di <nama-app>.herokuapp.com
